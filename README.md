Wordle - Java to Web Port
This project is an implementation of the popular game Wordle, originally developed as a console application in Java for my Web Development degree. To showcase my full-stack skills and growth, I have created a Web version so the game can be played directly in the browser, while also including the original backend logic.
Note: While the documentation is in English, the code comments in the Java backend remain in Spanish as they were part of the original academic submission.
